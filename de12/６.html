実際に作ったプログラム

用途：VScodeのGit Hubへの反映を自動化したい！
内容：
  • コマンドプロントを立ち上げる
  • HP更新のためのコマンドを打ち込む
  • コマンドプロンプトを終了する
この段階を目視しながら自動的に打ち込まれるようなシステム
  1. import pyautogui
  2. import time
  4. # コマンドプロンプトを開く
  5. pyautogui.press('win')
  6. pyautogui.write('komando')
  7. time.sleep(1)
  8. pyautogui.press('enter')
  9. time.sleep(2)
  11. # 文字入力
  12. pyautogui.write('cd mygit/xbp')
  13. pyautogui.press('enter')
  14. time.sleep(3)
  15. #読み込み時間のために数秒待つ
  16. pyautogui.write('git status')
  17. pyautogui.press('enter')
  18. time.sleep(3)
  19. pyautogui.write('git add .')
  20. pyautogui.press('enter')
  21. time.sleep(5)
  22. pyautogui.write('git commit -m "20211102"')
  23. pyautogui.press('enter')
  24. time.sleep(5)
  25. pyautogui.write('git push')
  26. pyautogui.press('enter')
  27. time.sleep(10)
  30. # コマンドプロントの終了
  31. pyautogui.hotkey('alt', 'f4')

参考：python でキーボード入力を自動化できるコードを応用して入力されるようにした
